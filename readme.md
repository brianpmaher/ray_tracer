# Ray Tracer

A toy ray tracer implementation in C.

## Setup

### Windows

1. [Install Visual Studio](https://visualstudio.microsoft.com/vs/) to get `vcvarsall.bat` script.
2. Run `build.bat` from win32 console

No setup for osX yet
